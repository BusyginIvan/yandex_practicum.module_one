# Blog Backend

Учебный проект: бэкенд веб-приложения блога.

Проект находится в активной разработке.  
В репозитории будет реализован REST API для управления постами и комментариями с использованием Java и Spring Framework (без Spring Boot).

Подробное описание архитектуры, API и способов запуска будет добавлено по мере развития проекта.
